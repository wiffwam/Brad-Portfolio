Battleship

Battleship is a client-server based game where two people on different computers can connect to a server, enter the same or different chat rooms, and challenge each other in a game of Battleship.

Software used: Netbeans IDE v8.2

Language:Java

Coding: I participated in all aspects in coding this project

Non-Coding related Tasks: creation of the documentation
