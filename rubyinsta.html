Instagramm

Instagramm is a Ruby Rails MVC web based application that allows logged in users to post pictures and comment and vote on pictures.

Text Edititor used used: Sublime Text 3

Language:Ruby, HTML, CSS

Gems Used/Outside Code Used:
Normalize.css - Renders all elements more consistently
Haml - Abstraction Markup Language for HTML
Simple_Form - Used to add forms to the application
Bootstrap-sass - Used to add bootstrap styling to the application
Paperclip - Upload mamagenet for ActiveRecord. As I was using a win pc I also had to install GNUwindows32.
Devise - Handles the user aspect of the application
