Ice Cream Shop

This application is a ASP.net based Ice cream application. While not pratical it demonstrates accessing a database using C# and ASPX

Software used: Visual Studio

Language:JavaScript, ASPX, SQL, C#

Application Break Down: This applicaiton consists of a drop down list box that is populated by pulling the data from a database. Depending on the selection, the information related to that product is displayed. A graph is also displayed on the page to represent ICe Cream Sales.
