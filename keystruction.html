Key-Struction Project

Key-Struction was a group project that was built around improving ones typing skills. The software is written using GameMaker Studio 2. This was a team project and I was listed as programming head.

Software used: GameMaker Studio v 2

Coding: I was responsible in writing the back end code for the main part of the game, as well front end aspects for the main game screen. A few examples of what I did in the project was: Code to handle the typing, key recognition, upper/lower case key recognition, and Sprites movement.

Non:Coding related Tasks: I was responsible for assigning tasks to the other programmer, and making sure that myself and the other developer stayed on schedule.

Q/A: I performed QA tests and documention upon the program
